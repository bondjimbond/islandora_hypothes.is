INTRODUCTION
------------
  This module embeds Hypothes.is into your site. Hypothes.is helps annotating
  the webpage.

  * For a full description of the module, visit the project page:
    https://www.drupal.org/project/hypothesis

  * To submit bug reports and feature suggestions, or to track changes:
    https://www.drupal.org/project/issues/hypothesis

REQUIREMENTS
------------
No special requirements

INSTALLATION
------------
  Install as you would normally install a contributed drupal module.
  @see https://www.drupal.org/documentation/install/modules-themes/modules-7
  for more details.


CONFIGURATION
-------------
  Visit admin/config/hypothesis for configuration.
